PROTOTYPAL INHERITANCE

Using Javascript

DEFINITION

Prototypal Inheritance are ways to definine object properties by creating a heirachry or classification system from objects that identify a property a paticular object inherits from another object of higher order... the term Objects Linked To Other Objects (OLOO) comes to mind

Understanding the basics of Prototypal Inheritence is a must of junior developers! So I will demonstrate this concept by linking objects with the Javascript syntax Object.create() and defining these objects properties.
